## 보고서 생성 방법

```bash
# 1. 최초 1회만 실행 (python-docx 설치)
uv venv .venv
uv pip install python-docx --python .venv/Scripts/python.exe

# 2. 매주 실행
.venv/Scripts/python.exe generate_report.py
```

> 출력: `Weekly foreign-macro_YYYYMMDD.docx` (맑은 고딕, A4)

*이 템플릿은 주간 해외매크로 보고서의 표준 양식입니다.*
